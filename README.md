# Ứng dụng Thời tiết bằng React

## Giới thiệu
Ứng dụng Thời tiết này được xây dựng bằng React, cho phép người dùng kiểm tra thông tin thời tiết hiện tại và dự báo trong tương lai cho bất kỳ địa điểm nào trên thế giới.

## Tính năng
- Hiển thị thời tiết hiện tại theo vị trí người dùng.
- Dự báo thời tiết trong 5 ngày.
- Tìm kiếm thời tiết theo thành phố.
- Tương tác thân thiện và dễ sử dụng.

## Công nghệ sử dụng
- React.js
- Redux cho quản lý trạng thái
- Axios để gọi API
- OpenWeatherMap API để lấy dữ liệu thời tiết

## Cài đặt
1. Clone repo: `git clone https://github.com/username/react-weather-app.git`
2. Cài đặt các dependency: `npm install`
3. Chạy ứng dụng: `npm start`
